# FastCoref.fs

Coreference resolution for English in pure .NET — F# port of the Python
[fastcoref](https://github.com/shon-otmazgin/fastcoref) library running on
[TorchSharp](https://github.com/dotnet/TorchSharp).

See the [README](https://github.com/et1975/FastCoref.fs#readme) for an overview,
installation, and usage examples.
